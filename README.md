# GeekBand-I150021

##团队项目
项目名称：蓦然

项目简介：“蓦然”App 是一个基于地理位 置的周边环境采集、发现、评论社区。使 用“蓦然”,用户可以对周边环境(建筑、 景点、街道、商铺.... )进行拍照,并发表 评论。也可以查看其他用户对周边环境拍 摄的照片和评论。
